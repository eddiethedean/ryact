# ryact-zod

[![PyPI](https://img.shields.io/pypi/v/ryact-zod.svg)](https://pypi.org/project/ryact-zod/)
[![Python](https://img.shields.io/pypi/pyversions/ryact-zod.svg)](https://pypi.org/project/ryact-zod/)
[![CI](https://github.com/eddiethedean/ryact/actions/workflows/ci.yml/badge.svg)](https://github.com/eddiethedean/ryact/actions/workflows/ci.yml)

Portable schema validation for the `ryact` ecosystem.

The goal is a **Zod-like authoring experience** with a **language-agnostic schema AST** so schemas can be shared between the TSX lane and Python lane.

## Install

```bash
pip install ryact-zod
```

## Status

Early scaffold: minimal AST + validator surface.

## Source of truth

- Inspiration: Zod (`https://github.com/colinhacks/zod`)
